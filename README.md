# Urban Task Manager

Aplikasi **Urban Task Manager** digunakan untuk mengelola dan mencatat tugas harian secara efisien.  
Aplikasi ini dibuat menggunakan **React.js** dan dapat dijalankan secara lokal di komputer.

## ðŸš€ Langkah Menjalankan Aplikasi Secara Lokal

### 1. Clone Repository
Buka **Git Bash** dan jalankan perintah berikut:
```bash
git clone <repo-url>
```

### 2. Masuk ke Direktori Proyek
Setelah proses clone selesai, masuk ke direktori proyek dengan perintah:
```bash
cd uts-react
```

### 3. Install Dependencies
Pastikan kamu sudah berada di direktori proyek, lalu jalankan perintah berikut di **Command Prompt (CMD)** atau **PowerShell**:
```bash
npm install
```

### 4. Jalankan Aplikasi
Setelah semua dependensi terpasang, jalankan aplikasi dengan:
```bash
npm run dev
```

### 5. Buka Aplikasi di Browser
Setelah perintah di atas dijalankan, buka browser dan akses:
```
http://localhost:3000
```
